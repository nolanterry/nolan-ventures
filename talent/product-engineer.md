# Product Engineer — Nolan Ventures

## The Role

Own the build and maintenance of 4 vertical SaaS products across the Nolan Ventures portfolio (Scian platform, FenceCalc, FireLog, CraneCheck). You'll ship features, fix bugs, manage deployments, and keep the codebase healthy — working async with high autonomy.

This is a hands-on IC role, not management. You build.

## What You'll Do

- Ship features and bug fixes across 4 Next.js apps (multiple repos)
- Manage Vercel deployments, preview environments, and production releases
- Write and maintain Playwright e2e tests for critical user flows
- Integrate third-party services (Stripe, Clerk, Sentry, PostHog, PDF generation)
- Own code quality — PR reviews, refactors, performance optimization
- Collaborate async via GitHub Issues, Telegram, and daily standups
- Respond to production bugs within 4 hours during working hours

## Tech Stack

- **Framework:** Next.js 14+ (App Router)
- **Language:** TypeScript
- **Auth:** Clerk
- **Database:** Neon (Postgres) + Drizzle ORM
- **Hosting:** Vercel
- **Payments:** Stripe
- **Styling:** Tailwind CSS + shadcn/ui
- **Testing:** Playwright
- **Monitoring:** Sentry, PostHog

## Required Skills

- 2+ years shipping production Next.js / React apps
- Strong TypeScript — not "I've used it," but "I think in it"
- Comfortable with Postgres, ORMs (Drizzle or Prisma), and schema migrations
- Experience with Vercel deployment workflows (preview deploys, env vars, domains)
- Can write Playwright or Cypress e2e tests
- Git proficiency — clean commits, branching, rebasing
- Self-directed — you don't need a Jira ticket to know what's broken

## Nice to Have

- Experience with Clerk auth
- Stripe integration (subscriptions, webhooks, customer portal)
- PDF generation (server-side rendering)
- Familiarity with vertical SaaS or compliance-driven industries
- Prior work in small/startup teams (< 10 people)

## Hours & Structure

- **20-30 hours/week** (part-time contractor)
- Async-first — overlap with ET business hours for standups/urgencies
- Flexible schedule, but responsive during ET working hours (9 AM - 6 PM)
- Ramp to full-time possible as portfolio grows
